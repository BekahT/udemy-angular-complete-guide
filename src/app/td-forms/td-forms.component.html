Template-Driven Forms Assignment Instructions:

  1. Add a Form with the following Inputs (and Validators)
    1. Email address (should not be empty and should be an email address)
    2. A Dropdown which allows the user to select from three different Subscriptions ("Basic", "Advanced", "Pro") Set "Advanced" as Default
    3. A Password field (should not be empty)
    4. A Submit Button
  2. Display a warning message if the Form is invalid AND was touched. Display a warning message below each input if it's invalid.
  3. Upon submitting the form, you should simply print the Value of the Form to the Console. Optionally, display it in your template.

Result:

Invalid email. Please try again.
Invalid password. Please try again.
At least one input contains invalid data or is empty.
