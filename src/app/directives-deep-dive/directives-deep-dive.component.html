Assignment Instructions:

  1. Create a directive to give a paragraph element a background color.
  2. Create a directive using Renderer2 to give a paragraph element a background color.
  3. Create a directive using HostListener that gives a paragraph a background color on the MouseEnter event.
  4. Create a directive using HostBinding that gives a paragraph a background color on the MouseEnter event.
  5. Create a directive that gives a paragraph a background color on the MouseEnter event and another background color on the MouseLeave event using the user's input for the background colors.

Result 1:

This paragraph is styled with a green background through the Basic Highlight directive.

Result 2:

This paragraph is styled with a blue background through the Renderer Highlight directive.

Result 3:

This paragraph is styled with a purple background on mouseenter through the Host Listener Highlight directive.

Result 4:

This paragraph is styled with a yellow background on mouseenter through the Host Binding Highlight directive.

Result 5:

This paragraph is styled with a background color, depending on the user's selection, by default and a different background color on mouseenter, also depending on the user's selection, through the User Input Highlight directive.
